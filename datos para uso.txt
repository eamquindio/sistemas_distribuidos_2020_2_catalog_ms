verificar el archivo aplication.properties y verificar o modificar los datos
para la coneccion ala base de datos root, password, nombre de base de datos

crear base de datos en postgreslq y despues darle clic y en restore se abre una ventana darle clic 
en los tres puntos donde dice Filename, se abre una nueva ventana en la parte de abajo donde dice format
seleccionamos allfiles y buscamos el archivo BD_API lo seleccionamos y por ultimo damos clic en restore
   

para consultar por store-id
get http://localhost:8080/Product_stores_discount/by-storeid?storeid={id}

para consultar por product-id
get http://localhost:8080/Product_stores_discount/by-productid?productid={id}

para consultar por id  en la api
get http://localhost:8080/Product_stores_discount/{id}

para consultar todos los datos en la api
get http://localhost:8080/Product_stores_discount


para crear datos en la api
post http://localhost:8080/Product_stores_discount
y le manda por body los siguientes datos

{
    "id":5,
    "discount_id":"1",
    "product_id":"819",
    "store_id":"1"
}

para actualizar los datos en la api
put http://localhost:8080/Product_stores_discount/{id}

y se le mandan los datos por el body en json
{
    "id":1,
    "discount_id":"1",
    "product_id":"819",
    "store_id":"1"
}


para borrar un dato en la api
delete http://localhost:8080/Product_stores_discount/{id}
